# MpsTransactionList Specification

## Overview
- **Target file:** `src/components/mps/MpsTransactionList.tsx`
- **Interaction model:** click-driven

## Features
- 全局搜索 + 7 个筛选字段（含账单邮箱）
- 表格列：操作、交易金额、订单状态、交易创建时间、商户订单号、PayKKa订单号、支付方式、卡号后四位、卡组、可退款、交易完成时间、场景
- 分页 10 条/页
